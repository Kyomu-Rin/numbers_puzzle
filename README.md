## 概要
数字を順番に消すだけのシンプルな暇つぶしゲーム  
⇒ [URLはこちら](https://play.google.com/store/apps/details?id=com.kyomurin.android.numberspuzzle&hl=ja)

## 遊び方
- タイトル画面でゲームを選択  
  - 4x4・5x5・6x6
  - NORMAL・HARD
- NORMALモード  
  - 数字を1から順番に消すだけ！
- HARDモード
  - **全ての数字の位置を暗記**    
    - 制限時間あり
  - 記憶を頼りに数字を消すだけ！
    - 一切数字が見えないので注意
    - 記憶力がアップするかも…？？
